# Branches

## Listing remote branches  
`git branch -r`

## Listing all branches (local and remote)  
`git branch -a`

## Delete a branch  
`git branch -d <branch_name`

## Copying a branch
`git branch -c [<oldbranch>] <newbranch>`

## Pushing a local branch to remote  
`git push (-u|--set-upstream) origin <branchname>
